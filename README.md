# obs-jisin

OBS Studio 用の地震情報テロップ表示ツール。P2P地震情報 API（気象庁データ）から地震情報を取得し、テレビ風のテロップとして表示します。

## セットアップ

1. このリポジトリをクローンまたはダウンロード
2. OBS Studio を開く
3. ソースの追加 → **ブラウザ** を選択
4. 以下を設定:
   - **ローカルファイル** にチェック
   - `index.html` のフルパスを指定
   - 幅: `1920` / 高さ: `1080`
   - **背景が透明なカスタムCSS** は空欄でOK
5. ソースの位置を画面下部に合わせる

## 機能

- 30秒ごとに地震情報を自動取得
- 過去24時間以内の地震情報をテロップ表示
- 震度に応じた色分け表示（震度1〜7）
- 津波情報の表示（注意報・警報）
- スライドイン/アウトのアニメーション
- 新しい地震情報のみ表示（重複排除）

## カスタマイズ

`script.js` 冒頭の定数で動作を調整できます:

| 定数 | 初期値 | 説明 |
|------|--------|------|
| `POLL_INTERVAL` | `30000` | API取得間隔（ミリ秒） |
| `MAX_AGE_HOURS` | `24` | 表示する地震の最大経過時間 |
| `DISPLAY_DURATION` | `15000` | テロップ表示時間（ミリ秒） |

## 震度カラー

| 震度 | 色 |
|------|-----|
| 震度1 | 水色 |
| 震度2 | 緑 |
| 震度3 | 黄 |
| 震度4 | オレンジ |
| 震度5弱 | 濃いオレンジ |
| 震度5強 | 赤 |
| 震度6弱 | ピンク |
| 震度6強 | 紫 |
| 震度7 | 濃い紫 |

## データソース

[P2P地震情報 API](https://www.p2pquake.net/) - 気象庁の地震情報を提供する公開API
